SOBRE MIM

Projetos

API REST FULL

Este projeto tem sido o mais desafiador de todos que já fiz, a ideia é criar uma API rest Full com node.js, express e JWT:


  • Para envios de testes utilizei o insomnia um framework open source para desenvolvimento;
  • O backend foi implementado a RESTful API com base em node.js, Express.js e MongoDB;
  • Neste momento estou desenvolvendo os Gets da aplicação.

Projetos

AGENDA PESSOAL

Um sistema de agenda pessoal para web com CRUD de contatos, com segurança e deploy dela:


  • O padrão de arquitetura foi MVC, com sistema de rotas com middlewares;
  • O banco de dados foi o MongoDB com mongoose além de criar um CRUD de contatos;
  • Utilizei POO e segui padrões de segurança como dotenv, hasing, salting, http;
  • Fiz deploy através do Git com chaves SSH e conectei ao servidor da Google Cloud através de VM.

LANCHO .NET

A Lancho .net é uma empresa fictícia, criada por meu grupo na faculdade, ela solicitou como TCC um sistema completo de uma empresa, desde a documentação até o deploy:
